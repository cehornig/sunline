Sunline is created and maintained by Claire Hornig. It's powered by Django, Pysolar, SunriseSunset.com, and the Plotly JavaScript Open Source Graphing Library.

The plot data is available as an API. You can access it without a date, using a latitude followed by a longitude:

https://www.sunline.com/api/40.5/-40.5
Or with a date:
https://www.sunline.com/api/date/40.5/-40.5/2020-06-21
If you use it, please include a link to this site, and please don't make excessive requests. Thanks!
